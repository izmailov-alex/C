task2_2_1

выводит значение функции f(x)=a_n*x^n + ... a_1*x + a_0 в точке x

task2_2_3

Выводит значение интеграла функции f(x)=a_n*x^n + ... a_1*x + a_0 в точке x

task2_3

task2_4
